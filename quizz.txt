Question 1 
Vous souhaitez envoyer une recherche avec des critères spécifiques pour filtrer les produits sur un site d’e-commerce. Quelle méthode HTTP est la plus adaptée ? 

    •    B) POST 



Question 2 
Un service Web vous permet de modifier le statut d’une commande de “en cours” à “livrée”. Quelle méthode HTTP serait la plus appropriée pour cette action ? 

    •    A) PATCH 



Question 3 
Vous construisez une API pour un jeu en ligne qui doit permettre aux joueurs de sauvegarder leurs scores après avoir terminé une partie. Quelle méthode HTTP devriez-vous utiliser pour cela ? 

    •    A) GET 



Question 4 
Dans le cadre d’une application de gestion de tâches, vous souhaitez permettre aux utilisateurs de récupérer les détails d’une tâche spécifique à partir de son identifiant unique. Quelle méthode HTTP utiliseriez-vous ? 

    •    B) GET



Question 5 
Vous développez une fonctionnalité permettant aux utilisateurs de retirer un article de leur panier d’achats sur une application mobile. Quelle méthode HTTP est la plus appropriée pour cette action ? 

    •    D) DELETE
